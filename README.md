# PolyOS
Super duper simple OS that does nothing special and is currently not very useable due to its lack of actual OS features like file system, user mode, etc.

## Why is this a thing?
It's mainly for me (AAli107) to learn, and to see how far can I go with OS Development. i don't expect to go far.
